# Antigravity Traces

This directory contains agent traces generated during development using Google Antigravity. 
These traces log the inputs, reasoning, and outputs of each agent in the system to help debug multi-agent interactions.

## Sample Trace Scenario: MI vs CSK, Over 17 (Chasing 196, 152/5)

- **Round 1 (Proposal)**: Strategist recommends bringing back Pathirana despite economy, and using timeout.
- **Round 2 (Challenge)**: Devil's Advocate suggests Jadeja instead to target Hardik, defer timeout.
- **Round 3 (Defense)**: Strategist points out Jadeja's weakness on flat tracks, defends Pathirana and timeout timing.
- **Round 4 (Final)**: Moderator approves Pathirana for the death but adjusts timeout timing based on debate.
- **Round 5 (Commentary)**: Commentator translates the tactical decision to broadcast-ready narrative.
